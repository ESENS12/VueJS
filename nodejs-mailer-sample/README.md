 - simple mailer sample (using node-mailer) 

# References

# Project setup
```
mkdir nodejs-mailer-sample && cd nodejs-mailer-sample
```
```
npm init
```
```
npm install express --save
```
```
npm install nodemailer
```
```
cd nodejs-mailer-sample && node index.js
```
