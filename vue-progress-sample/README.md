# vue-progress-sample
 - Simple progress dialog samples

# References
 - K-Progress  (https://github.com/xrkffgg/k-progress)
 

# Project setup
```
vue create vue-progress-sample && cd vue-progress-sample && add vuetify
```

```
yarn add k-progress
```

```
yarn install && yarn serve
```
